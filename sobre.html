Explicação sobre a Disciplina

A disciplina de Desenvolvimento Web tem como maior foco o Front-end, ou seja, a camada da aplicação que roda no navegador do cliente. Enquanto as disciplinas de banco de dados e programação lidam com o Back-end, os "bastidores" do sistema, o desenvolvimento web é onde se cuida de tudo o que acontece visivelmente.

Nesta disciplina, o objetivo não é usar construtores de sites prontos, mas sim entender a engenharia por trás do navegador. Você aprenderá a escrever o código que os navegadores como o Chrome, interpretam para renderizar páginas visuais.

HTML: O esqueleto da web, onde se pode estruturar conteúdos, utilizando textos, listas e tabelas, conectar páginas através de links e navegação, além de capturar dados do usuário com a criação de formulários e inputs.

CSS: Onde há a criação e aplicação de estilos e onde pode-se dominar ferramentas como o Grid Layout para posicionar elementos em tela, transições e animações para tornar a interface mais agradável.

JavaScript: Introduz a programação à página web, passando por variáveis, tipos de dados, laços de repetição e funções, manipulação do DOM para reagir a eventos do usuário, tornando a interface dinâmica, e utilização de arrays e objetos para gerenciar dados.

Bootstrap: Um dos frameworks mais populares do mercado, possibilita utilizar o sistema de grid do Bootstrap, sua tipografia, cores e componentes pré-construídos para acelerar o desenvolvimento e garantir que suas aplicações sejam responsivas.

Por que é importante?

Um sistema back-end pode ser incrivelmente rápido e ter um banco de dados perfeitamente otimizado, mas se a interface web for confusa, feia ou não funcionar no celular, o projeto pode fracassar. O usuário final julga o sistema pela tela que ele vê e interage. Além disso, o mercado de tecnologia valoriza imensamente profissionais que conseguem transitar entre o Back-end e o Front-end. Dominar HTML, CSS e JavaScript é o requisito fundamental para se tornar um desenvolvedor Full-Stack, abrindo um leque muito maior de oportunidades de emprego e projetos.

Na arquitetura de software moderna, o Back-end frequentemente expõe dados através de APIs. É o Front-end que fará as requisições a essas APIs, recebendo os dados brutos e transformando-os em tabelas, gráficos e painéis visuais para o usuário. Diferente de softwares instalados localmente, uma aplicação web pode ser acessada de qualquer lugar do mundo, por qualquer dispositivo com um navegador.

Aprender a criar layouts responsivos garante que o seu software seja universalmente acessível, seja em um monitor de 32 polegadas ou na tela de um smartphone antigo. Do ponto de vista do aprendizado, o Desenvolvimento Web oferece uma recompensa visual instantânea. Mudar uma linha de código em CSS ou JavaScript altera imediatamente o que você vê na tela. Isso ajuda a consolidar a lógica de programação de forma mais tátil e engajadora.
