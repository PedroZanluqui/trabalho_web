Conteúdos Estudados

Clique em cada tópico para ver os detalhes.

HTML
  • Estruturação de Conteúdo: Uso correto das tags de texto, listas, e formatações visuais.
  • Links e Navegação: Criação de hiperlinks, entendimento de atributos, e mapeamento de links internos e externos.
  • Mídia e Dados Tabulares: Inserção de imagens e criação de tabelas.
  • Coleta de Dados: Estrutura básica de Formulários e Inputs.
  • Organização de Páginas: Uso de frames e integração inicial com folhas de estilo.
CSS
  • Introdução ao CSS: Como o CSS funciona, criação e tipos de estilos.
  • Estruturação de Páginas: Técnicas de posicionamento e o uso do poderoso sistema de Grid Layout.
JavaScript
  • Fundamentos da Programação: Variáveis, Tipos de Dados, Operadores e Estruturas Condicionais.
  • Estruturas de Controle e Organização: Uso de Laços de Repetição (Loops) e criação de Funções.
  • Estrutura de Dados: Manipulação e armazenamento de dados utilizando Arrays e Objetos.
  • Dinamismo em Tela: Manipulação do DOM para alterar a página via código.
Bootstrap
  • Layout Responsivo: Domínio do Sistema de Grid do Bootstrap para adaptação em diferentes tamanhos de tela.
  • Identidade Visual Padrão: Uso das classes utilitárias para Tipografia e Cores.
  • Formulários e Inputs no Bootstrap: Aplicação de estilos avançados e padronizados para coleta de dados.
  • Componentes de Navegação: Implementação de Menus e trilhas de navegação.
