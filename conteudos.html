Conteúdos Estudados

Clique em cada tópico para ver os detalhes.

HTML5 — Estrutura e Semântica
O HTML5 é a base de toda página web. Aprendemos a usar tags semânticas como header, nav, main, section e footer para estruturar documentos corretamente.
CSS3 — Estilização e Layout
O CSS3 cuida da aparência visual. Estudamos seletores, box model, Flexbox, Grid Layout e media queries para criar interfaces responsivas.
Bootstrap — Framework CSS
O Bootstrap é um framework que acelera o desenvolvimento com componentes prontos como navbar, cards, grids e formulários responsivos.
JavaScript — Interatividade
O JavaScript adiciona comportamento dinâmico às páginas. Estudamos variáveis, funções, eventos, manipulação do DOM e validação de formulários.
